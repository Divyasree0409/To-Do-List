# ✅ To-Do List Web Application

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.  
This project helps users manage their daily tasks efficiently by allowing them to add, complete, and delete tasks.

---

## 📌 Features
-  Add new tasks  
-  Mark tasks as completed  
-  Delete tasks  
-  Display task list dynamically  
-  (Optional) Save tasks using local storage  

---

##  Technologies Used
- HTML5
- CSS3
- JavaScript
